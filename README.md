# Film Tracker — консольное приложение для учёта просмотренных фильмов

Небольшое консольное приложение для ведения списка фильмов: добавлять фильмы, отмечать просмотры и искать по году.

---

## Возможности
- Добавление фильма (название, год).
- Отмечать просмотр.
- Поиск фильмов по году.

---

## Требования
- Python 3.8+ (рекомендуется).
- Зависимости отсутствуют (чистый Python) — если в проекте есть requirements.txt, установите `pip install -r requirements.txt`.

---

## Быстрый старт
1. Убедитесь, что установлен Python 3.
2. Запустите приложение:

```bash
python app.py
```


---

## Тесты
Для запуска тестов используйте:

```bash
pytest
```

Если в проекте нет `pytest` в окружении — установите его:

```bash
pip install pytest
```

---


## Структура проекта
```
film-tracker/
├── app.py          # входная точка приложения
├── logic.py        # модель и функции для работы со списком фильмов
├── movies.json    # пример файла хранения (генерируется приложением)
├── tests/          # тесты pytest

```

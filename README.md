Консольное приложение для ведения списка просмотренных фильмов.

Позволяет добавлять фильмы, отмечать просмотры, ставить оценки и искать по году.

1. Убедитесь, что установлен Python 3.
2. Запустите приложение:
   ```bash
   python app.py
   
Для проверки работоспособности программы запустите тесты:
   ```bash
   pytest
